## 📚 Conceitos Fundamentais

### 🅰️ O que é um Projeto?

Um **projeto** é um conjunto de atividades realizadas com o objetivo de criar:

* um produto
* um serviço
* ou um resultado específico

\---

### 🅱️ O que é um Programa?

Um **programa** é um conjunto de projetos relacionados, com o objetivo de gerar um resultado maior.

📌 Exemplo:

* Um app mobile evoluindo para um sistema web completo

\---

### 🅲 O que é um Portfólio?

Um **portfólio** é o conjunto de programas e projetos que:

* não precisam estar diretamente relacionados
* mas geram valor para a organização

🎯 Foco:

* Estratégia
* Marketing
* Crescimento

\---

### 🔺 O que é o Triângulo do Projeto?

O triângulo representa três restrições principais:

* **Escopo** → O que será feito
* **Tempo** → Prazo disponível
* **Custo** → Orçamento disponível

📌 Importante: alterar um desses afeta os outros.

\---

### 🔄 Modelos de Processo

Forma fácil de memorizar: **A.E.I.C**

|Letra|Modelo|Descrição|
|-|-|-|
|A|Ágil|Flexível, iterativo e rápido|
|E|Espiral|Baseado em análise de riscos|
|I|Incremental|Entregas em partes|
|C|Cascata|Sequencial, fase por fase|

\---

### ⚙️ O que é Scrum?

O **Scrum** é um framework ágil focado em:

* colaboração
* organização
* entregas contínuas

\---

## 💡 Valores do Scrum (2CFA)

Forma fácil de memorizar:

* **Coragem** → Enfrentar desafios
* **Compromisso** → Foco no objetivo
* **Foco** → Priorizar o que importa
* **Abertura** → Transparência e comunicação
