📌 Sobre o Projeto



Este repositório foi desenvolvido com o objetivo de centralizar e organizar conteúdos da disciplina de Gerenciamento de Projetos, servindo como um ambiente de aprendizado contínuo.



A proposta não é apenas armazenar materiais, mas também criar um espaço para:



📈 Evolução constante

🤝 Troca de conhecimentos e ideias

🧠 Consolidação dos principais conceitos

🛠️ Aplicação prática do conteúdo

🎯 Objetivos

Reunir conteúdos relevantes da disciplina

Facilitar revisões futuras

Desenvolver uma base sólida em gerenciamento de projetos

Aplicar conceitos em cenários reais
